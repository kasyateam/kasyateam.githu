Комментарии в Sass

Комментарии в стилевых файлах предназначены для пояснения фрагментов кода и заметок разработчика.

Комментарии никак не влияют на сам код и они игнорируются браузерами.

Sass поддерживает два вида комментариев, условно их назовём многострочными и однострочными.

Многострочные комментарии

Комментарием считается любой текст между /* и */.

На деле, конечно, это может быть как одна строка, так и несколько.

/*
    Комментарии в стиле Drupal
*/

.sidebar
{
    background: #f0f0f0;
}

Такой комментарий напрямую переносится в CSS-файл, поскольку стилевые файлы поддерживают аналогичный синтаксис комментариев.

Однострочные комментарии

Комментарием считается любой текст после //.

Такой вид комментариев характерен для некоторых языков программирования, например, JavaScript.

.sidebar
{
        background: #f0f0f0; // Цвет фона
    color: #800; // Цвет текста
}

Однострочные комментарии в CSS-файл не переносятся.

При компиляции мы получим следующий стиль.

.sidebar
{
    background: #f0f0f0;
        color: #800;
}
