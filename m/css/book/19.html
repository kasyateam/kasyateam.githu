Теперь Вы знаете CSS, что дальше?

После изучения CSS Вы можете переходить к изучению CSS3 или JavaScript.

CSS3

CSS3 - это обновленная версия CSS. В CSS3 было добавлено большое количество новых возможностей, таких как:

- Создание анимации;

- Возможность добавления к элементам и тексту элементов теней;

- Использование на веб-страницах произвольных шрифтов (а не только безопасных);

- Создание градиентов и многое другое.

Теперь Вы знаете CSS и HTML и следовательно можете создавать статические страницы и оформлять их.

Такие страницы хорошо подходят для передачи обычного текста.

Но если Вы хотите "оживить" страницы, добавить в них динамику и позволить пользователю взаимодействовать с ними, то Вам просто не обойтись без JavaScript.

JavaScript

С помощью JavaScript Вы можете превратить статические страницы созданные с помощью HTML и CSS в динамические страницы способные взаимодействовать с пользователем (выводить окна с сообщениями, изменять свое содержимое после определенных действий и т.д.).

На данный момент JavaScript является самым популярным скриптовым языком в интернете и он поддерживается всеми современными браузерами.
