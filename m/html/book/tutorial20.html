Что делать дальше?

Теперь когда Вы знаете HTML - фундамент всемирной паутины, Вы можете переходить к изучению HTML5, XHTML, CSS или JavaScript.

HTML5

HTML5 это новая версия HTML.

В HTML5 было добавлено много нового, поэтому некоторые возможности, которые были доступны в HTML4 только с использованием внешних плагинов (таких как Adobe Flash) или клиентских скриптов теперь доступны с помощью обычных разметочных тегов.

CSS

CSS используется для оформления HTML документов.

Благодаря CSS все оформление может быть удалено из HTML документов и может храниться в отдельном файле.

JavaScript

С помощью JavaScript Вы можете создавать динамические сайты и веб-приложения.

Динамические сайты в отличие от статических (т.е. созданных с использованием только HTML) могут взаимодействовать с пользователем, к примеру после нажатия на кнопку пользователю может быть отображено определенное сообщение.
