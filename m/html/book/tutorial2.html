Создание HTML документов

Для того, чтобы создать HTML документ необходимо:

1. Открыть любой текстовый редактор (например блокнот встроенный в Windows);

2. Набрать произвольный текст и разметить его HTML тегами;

3. Cохранить файл с расширением .htm или .html.

Теперь если Вы откроете созданный файл с помощью веб-браузера он будет отображен как веб-страница.

Редакторы HTML документов

Несмотря на то, что для создания HTML документов Вы можете использовать даже стандартный текстовый редактор (например блокнот в Windows) мы не рекомендуем делать этого.

Для этих целей создано множество специальных редакторов с подсветкой тегов и другими полезными возможностями.

Примеры таких редакторов:

- Notepad ++

- Microsoft FrontPage

- Adobe Dreamweaver

- Visual Studio Code (Лично моё предпочтение)

Какое расширение лучше использовать .htm или .html?

Это не имеет значения, так как все современные браузеры одинаково хорошо понимают и .htm и .html.

Мы предпочитаем использовать расширение .html так как считаем, что оно наиболее наглядно, но конечный выбор конечно остается за Вами.
