HTML элементы

HTML элементом называется комбинация начального тега, конечного тега и содержимого.

Примеры HTML элементов:

Начальный тег Содержимое элемента Конечный тег
<p> Это абзац. </p>
<b> Это жирный текст. </b>

Большинство элементов могут также иметь атрибуты.

О конечном теге

Браузер верно отобразит HTML элемент даже если Вы забудете указать конечный тег.

HTML является языком "прощающим ошибки", однако существует и более строгая разновидность языка (XHTML), в которой пропуск конечного тега будет считаться ошибкой.

<p>Параграф без закрытого тега
<p>Еще один параграф без закрытого тега
