HTML Введение

HTML расшифровывается HyperText Markup Language (в переводе означает Язык Разметки ГиперТекста).

Для изучения HTML Вам не потребуется устанавливать на компьютер дополнительное программное обеспечение. Вы сможете создавать веб-страницы с помощью встроенного в сайт онлайн-редактора.

HTML предназначен для создания веб-страниц во всемирной паутине.

HTML настолько легко изучить, что уже через несколько минут Вы сможете создать свою первую веб-страницу.

HTML Теги

Любой HTML документ состоит из обычного текста и разметочных тегов.

Текст как и в любых других файлах используется для передачи информации, а разметочные теги используются для ее группировки и оформления.

Разметочные теги HTML представляют собой специальные слова, которые окружены с обеих сторон угловыми скобками, например: <html>.

HTML теги обычно используются в паре, например: <b></b>.

Первый тег называется начальным тегом, а второй конечным тегом.

Текст находящийся между начальным и конечным тегом подвергается "разметке".

Например <b>Привет</b> будет отображено браузером как Привет (слово 'Привет' написанное жирным шрифтом).

HTML документ

Если HTML документ будет открыт в редакторе он будет отображен как обычный текстовый файл.

Если HTML документ будет открыт в браузере он будет отображен в соответствии с разметочными тегами и будет называться веб-страницей.

Главная цель любого веб-браузера (такого как IE, Firefox, Chrome и т.д.) прочитать HTML документ и отобразить его как веб-страницу.

Верхняя часть редактора отображает HTML документ как текстовый файл, а нижняя отображает его как веб-страницу:

<html>
	<body>
		<h1>Мой первый заголовок!</h1>
		<p>Привет всем!</p>
	</body>
</html>
Результат работы кода:

Мой первый заголовок!

Привет всем!

Объяснение примера:
- Тег <html> объявляет начало и конец HTML документа;
- Тег <body> содержит все видимое содержимое HTML страницы;
- Тег <h1> позволяет "разметить" вложенный в него текст как заголовок;
- Тег <p> позволяет "разметить" вложенный в него текст как абзац.

Обратите внимание: в этом курсе вы не только сможете ознакомиться с правилами, но и увидеть примеры работы структуры на HTML
