HTML

HTML расшифровывается HyperText Markup Language (в переводе означает Язык Разметки ГиперТекста).

был основан в 1993 году

CSS

CSS расшифровывается Cascading Style Sheets (Каскадные Документы Стилей).

был основан в 1996 году

JavaScript

С помощью JavaScript Вы сможете создавать интерактивные веб-страницы.

был основан в 1995 году

PHP

PHP (англ. PHP: Hypertext Preprocessor — "PHP: препроцессор гипертекста") — скриптовый язык программирования общего назначения, интенсивно применяемый для разработки веб-приложений.

был основан в 1995 году
